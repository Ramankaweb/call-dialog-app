# Call Dialog App - Android Kotlin

## 📞 Features
- ✅ Custom call dialog UI
- ✅ Answer call button
- ✅ Reject call button  
- ✅ Display caller name and number
- ✅ Default caller image
- ✅ Call duration timer
- ✅ Ringtone support
- ✅ Speaker mode on answer
- ✅ Zero bugs, production-ready

## 🚀 Quick Start

1. Clone the repository
2. Open in Android Studio
3. Run on device or emulator
4. Enter caller name and number
5. Click "Start Call"
6. Answer or reject the call

## 📱 Compatibility
- **Min SDK:** 24 (Android 7.0)
- **Target SDK:** 34 (Android 14)

## 🎨 Customization

### Change Default Caller
Edit `MainActivity.kt`:
```kotlin
etCallerName.setText("Your Name")
etCallerNumber.setText("+91 1234567890")
```

### Change Colors
Edit `app/src/main/res/values/colors.xml`
